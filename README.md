# Datathon
Efetuar o download das bibliocas e suas devidas versões para o uso do notebook e app, seguindo o arquivo requirements.<br>
Caso vá ser usado o Anaconda Prompt, segue arquivo Instrucoes.txt para prosseguir no mesmo.<br>
Executar notebook para criação dos modelos, após isso executar pelo CMD 'streamlit run app.py' na pasta local onde se encontra app para abrir o Streamlit localmente e interagir com o App. <br>
  
Se possivel para avaliação rodar o Streamlit Local pois por varias tentativas com durações de 30 minutos cada a plataforma não consegui fazer o deploy dos arquivos, travando nos downlods de requerimentos.
